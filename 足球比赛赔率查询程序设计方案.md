# 足球比赛赔率查询程序 - Agent Team 方案

## 项目概述

构建一个足球比赛赔率查询程序，支持查询实时赔率、历史数据、多家博彩公司对比。

---

## Team 成员安排

### Team Lead — Claude Sonnet 4.6（你）
- 职责：任务拆分、架构设计、代码整合、Code Review、最终 PR
- 负责模块：项目骨架、模块接口定义、最终集成

---

### 队友 1 — API Researcher（数据源研究员）
- 职责：调研可用的足球赔率 API（如 The Odds API、Betfair API、OddsPortal 等），确定数据源方案，输出 API 接入文档
- 技能：Web 搜索、API 文档分析
- 交付物：`docs/api-research.md`（API 选型报告 + 接入示例）

---

### 队友 2 — Backend Developer（后端开发）
- 职责：实现数据获取层，封装 API 调用、数据解析、缓存逻辑
- 技能：Python/Node.js、HTTP 客户端、数据处理
- 负责文件：
  - `src/api_client.py` — API 请求封装
  - `src/data_parser.py` — 数据解析与标准化
  - `src/cache.py` — 本地缓存（避免频繁请求）

---

### 队友 3 — CLI/UI Developer（界面开发）
- 职责：实现命令行交互界面，支持按联赛/球队/日期查询，格式化输出赔率表格
- 技能：CLI 框架（Click/Typer）、Rich 表格渲染
- 负责文件：
  - `src/cli.py` — 命令行入口
  - `src/formatter.py` — 赔率表格格式化输出

---

### 队友 4 — QA & Config Engineer（测试与配置）
- 职责：编写测试用例、配置管理（API Key 等环境变量）、项目打包
- 技能：pytest、dotenv、项目配置
- 负责文件：
  - `tests/` — 单元测试
  - `.env.example` — 环境变量模板
  - `pyproject.toml` / `requirements.txt` — 依赖管理

## 核心功能

1. 查询指定联赛当日比赛赔率（主胜/平/客胜）
2. 多家博彩公司赔率对比
3. 按球队名称搜索
4. 历史赔率查询（如 API 支持）
5. 结果导出为 CSV

---

## 启动命令示例

```bash
# 查询中超今日赔率
python -m football_odds query --league chinese_super_league

# 查询指定球队
python -m football_odds search --team "上海海港"

# 对比多家博彩公司
python -m football_odds compare --match-id 12345
```
